# 🏋️ SmartGym - AI-Powered Gym Management System

A full-stack web application for modern gym management with AI-powered features.

## Quick Start

```bash
# 1. Create virtual environment
python -m venv venv
venv\Scripts\activate      # Windows

# 2. Install dependencies
pip install -r requirements.txt

# 3. Setup database
python setup_db.py

# 4. Run the app
python run.py
```

Open: http://localhost:5000

## Features
- AI Daily Workout Plans
- Indian Diet Planner (Budget-Friendly)
- Calorie Tracker
- QR-Based Attendance
- Membership & Fees Management
- Streak System
- Progress Dashboard
- AI Fitness Chatbot
- Leaderboard
